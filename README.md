# What-beats-rock

A remake of the https://www.whatbeatsrock.com/ game using Meta's llama 3.2

### Installation

This project uses [Poetry](https://python-poetry.org/) to manage dependencies. Make sure you have it
installed if you haven't already
from [here](https://python-poetry.org/docs/#installing-with-the-official-installer)

To install all dependencies and create a virtual environment, run

```
poetry install
```

Then to build the package,

```
poetry build
```

This will create the `.whl` file to install in the `dist` directory
